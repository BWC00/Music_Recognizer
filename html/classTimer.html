double Timer::elapsedMilliseconds ( )

Fetches the elapsed milliseconds. If the timer is still running, it will assume the time of this function call as the end of the timed period, it will not stop the timer. If the timer is no longer running, the elapsed milliseconds from the last start to the last stop are returned.
